Pasting into diagram.net with a transparent background

The cleanest path is the draw.io library file shipped alongside this page: in diagram.net open File → Open Library… and pick zscaler-stencils.drawio-library.xml. The shapes appear in the left panel — drag any one onto the canvas. They're transparent SVG, so no background comes with them. In this page you can also drag a shape straight onto the draw.io canvas, or use Copy PNG / Download. If copy does nothing, open this file in a real browser tab (the clipboard is blocked inside preview panes).

Preview: dark
How to use.  diagram.net / draw.io: open the supplied .drawio-library.xml via File → Open Library, then drag shapes from the panel (transparent, no background). Or drag a shape from this page straight onto the canvas.  PowerPoint / Word / Lucid: Copy PNG and paste, or drag the shape in.  Figma / web: Copy SVG, or use the .svg files.  Visio: open the .vsdx (desktop Visio) and copy shapes, or Insert → Pictures with any .svg.
